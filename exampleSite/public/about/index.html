About This Demo

This is the TUI Nav Hugo theme - a terminal-inspired, keyboard-navigable theme.

Keyboard Navigation

Use your keyboard to navigate:

Key Action
j or ↓ Move down
k or ↑ Move up
Enter or → Open item
← Go back
Escape Home

Features

  • Carousel view showing 5 items at a time
  • Terminal aesthetic with blinking cursor
  • Animated title letters
  • Code syntax highlighting
  • Responsive design
  • System monospace fonts

Code Example

function greet(name) {
  console.log(`Hello, ${name}!`);
}

greet('World');

Try It

Press ← to go back to the homepage, then use j/k to navigate between sections.
