On Simplicity

January 8, 2025 #essay

In an age of infinite complexity, simplicity is a radical act.

Every line of code is a liability. Every feature is maintenance. Every option is cognitive load.

The best interfaces disappear. They don’t ask you to think about them. They just work.

A terminal prompt. A blinking cursor. An empty page.

These are invitations, not obstacles. They say: you’re in control.

The modern web forgot this. Pop-ups asking for notifications. Cookie banners. Newsletter modals. Infinite scroll designed to trap you.

What if we went back? What if a website was just… words?
